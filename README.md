# angularjs-livechat
AngularJS component to integrate LiveChat with your single-page app

## Install
```
npm install --save angularjs-livechat
```

## Usage

Add to app dependencies:
```js
angular.module('app', ['ngLiveChat']);
```

Use directive:
```html
<livechat license="007"></livechat>
```
